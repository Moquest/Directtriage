# DirectTriage Website

Professionele triage, waarneming en consultatie voor commerciële zorgklinieken.

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Netlify (hosting & forms)

## Local Development

1. Clone repository
2. Open `index.html` in browser
3. No build process needed

## Deployment

### Via Netlify:
1. Push to GitHub
2. Connect repository in Netlify
3. Auto-deploy on push to main

### Build Settings:
- Build command: (leave empty)
- Publish directory: `/`

## Domain Setup

Add custom domain `directtriage.nl` in Netlify:
- A record: `75.2.60.5`
- CNAME www: `[your-site].netlify.app`

## Contact Form

Uses Netlify Forms (free). Form submissions appear in Netlify dashboard.
